## Endpoints

1. The app.post() endpoint with path "/patients" allows a user to create a patient object.

2. The app.put() endpoint with path "/patients/:id" allows a user to update a patient object by id.

3. The second app.put() endpoint with path "/patients/:id" allows a user to update specific details of the patient object.

4. The app.get() endpoint with path "/patients" allows a user to view all patient objects.

5. The app.get() endpoint with path "/patients/:id" allows a user to view a patient object by id.
